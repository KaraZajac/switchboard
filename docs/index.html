Switchboard

A modern IRC client with a familiar, Discord-like interface. Built with Electron and packed with IRCv3 support.

Download Source Code

Features

Everything you need from an IRC client, with a modern twist

🌐

Multi-Server

Connect to as many IRC networks as you want, all at once. Each server gets its own icon in the sidebar.

💬

Rich Messaging

Replies, reactions, typing indicators, read markers, message editing, deletion, and full IRC color/formatting support.

🎨

GIF Picker

Search and send GIFs, stickers, clips, memes, and emoji via Klipy — right from the message composer.

📎

File Uploads

Upload files via draft/filehost. Images and video render inline; other files get a download card.

🔗

Link Previews

OpenGraph metadata cards for shared URLs with title, description, and thumbnail — fetched automatically.

👥

Friend List

Track when your friends are online or offline using IRCv3 MONITOR. Persists across restarts.

👤

User Profiles

Edit your nickname, realname, and avatar. View other users' WHOIS info with a click.

📅

Chat History

Messages stored locally in SQLite and fetched from the server via draft/chathistory. Never miss a conversation.

🔎

Search

Full-text search across all your messages. Quick channel switcher with Ctrl+K.

🔇

Muting

Timed or permanent mute for channels and servers. No more notification spam from noisy channels.

🎨

13 Themes

Catppuccin, Dracula, Nord, Gruvbox, One Dark, Rosé Pine, Solarized, Tokyo Night, Kanagawa, Discord, and more.

💻

Cross-Platform

Available for macOS (.dmg), Linux (.AppImage, .deb, .rpm), and Windows (.exe) with auto-updates.
